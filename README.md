# Class GPA Calculator 🎓

A simple, real-world Python application that reads student grades from a CSV file, calculates individual GPAs based on a 4.0 scale, and computes the overall class average.

##  Features
- Reads data from a `.csv` file.
- Maps letter grades (A-F) to numerical values.
- Calculates and formats GPAs to two decimal places.
- Provides a summary of the class average.

1. **Prepare your Data**: Create a file named `class_grades.csv` in the same directory as the script.
